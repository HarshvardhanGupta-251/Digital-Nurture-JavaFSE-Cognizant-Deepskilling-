## 🚀 Getting Started

### Prerequisites

- Java Development Kit (JDK) 8 or higher
- Maven or Gradle
- Git
- An IDE (IntelliJ IDEA, Eclipse, or VS Code)

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/HarshvardhanGupta-251/Digital-Nurture-JavaFSE-Cognizant-Deepskilling-.git
   cd Digital-Nurture-JavaFSE-Cognizant-Deepskilling-
   ```

2. **Set up your Java environment**:
   - Ensure Java is installed: `java -version`
   - Set JAVA_HOME environment variable

3. **Build the project**:
   ```bash
   mvn clean build
   # or with Gradle
   gradle build
   ```

4. **Run the application**:
   ```bash
   mvn spring-boot:run
   # or
   gradle bootRun
   ```
