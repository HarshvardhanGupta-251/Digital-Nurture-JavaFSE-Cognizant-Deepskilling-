| Operation      | HashMap Complexity |
| -------------- | ------------------ |
| Add Product    | O(1)               |
| Update Product | O(1)               |
| Delete Product | O(1)               |
| Search Product | O(1)               |


Optimization

Operations can be optimized by:

. Using a good hash function.
. Choosing an appropriate initial capacity.
. Avoiding unnecessary traversals.
. Using TreeMap if sorted data is required.
